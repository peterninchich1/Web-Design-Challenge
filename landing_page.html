Climate Analysis Summary

The goal of this analysis is to identify fluctuations in weather patterns relative to the locations distance from the equator. The analysis began with calling a request to the OpenWeatherMap API to retrieve relevant data.

Below are links to all visualizations.

After using Pandas and Python to create a dataframe, Matplotlib was used to plot various graphs. We considered temperature, cloudiness, wind speed, and humidity. This site visualizes these results and summarizes the correlations.
